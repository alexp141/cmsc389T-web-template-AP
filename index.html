Projects

Recipe App

  • Written in Swift using frameworks such as SwiftUI, Firebase, and Firebase Authentication that connects to a realtime database.
  • Users can create an account, upload their own recipes (including photos), and browse recipe posts submitted by other users.
  • Users can filter recipes using a search bar and save recipes to their favorites list.
  • Users can follow each other and rate anyone's recipes.

Path Tracking App

  • Built using the SwiftUI and Mapkit frameworks.
  • With a push of a button, the app tracks the user's current location and continuously records their current route using map annotations.
  • Uses Apple's FileManager API to efficiently save recorded tracks to the user's local storage.

BlackJack Simulation

  • A Blackjack game built using Java
  • fully functional GUI constructed with Java Swing.
  • Allows options for betting and changing betting amounts.
